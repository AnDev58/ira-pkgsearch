About IRA Origin

IRA Origin is a place where everybody can publish his IRA packages. This repository is maintained by creators of IRA and IRA community. So, if your package is rather good, everybody will have ability to install it really easy.

Publishing packages

As the developer, you can upload your package as IPKG (compiled program) or repository (sources) From this moment, your package will be availible for everybody

Dangerous, Safe and Verified

After package has been uploaded, it will be marked as Dangerous. To become safe, package must have more than 50 community reviews. Also, if one of moderators will check your program, it becomes Verified

Community reviews

If you want to help, but you wouldn't like to write application, you may do community review. It means testing, writing review mark and reviewing code if program is open-source.

About 3rd-party

Third-party software also can be uploaded here, but you will need to get a premium account. That means you need to pay for uploading third-party software. Also, to become verified, you should send sources to moderator, if it's necessary.
